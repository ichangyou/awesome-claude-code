# Contributing to Awesome Claude Code

Thank you for contributing! This list is community-maintained and only stays useful if entries are high quality.

## Adding an Entry

1. **Fork** this repository and create a new branch.
2. Add your entry to the appropriate section in `README.md`.
3. Use this format exactly:

```
- [Name](URL) — One-line description starting with a capital letter, ending without a period
```

4. Submit a **Pull Request** with a brief note on why you're adding it.

### Optional flags (append after description)

| Flag | Meaning |
|------|---------|
| `` `(Official)` `` | Maintained by Anthropic or the MCP team |
| `` `(Chinese)` `` | Chinese-language resource |
| `` `(Paid)` `` | Requires payment or subscription |

## Quality Bar

Before submitting, confirm:

- [ ] You have personally used this tool/resource and found it valuable
- [ ] The link is live and points to the correct page
- [ ] The description is specific — not just "a great tool for Claude Code"
- [ ] The entry fits an existing section (if not, propose a new section in the PR)
- [ ] No duplicate of an existing entry

## What We Don't Accept

- Unmaintained repos (no commits in 12+ months, unless historical value)
- Self-promotion without genuine value to the community
- Generic AI tools not specifically useful with Claude Code
- Paywalled content without a clear free tier or trial

## Reporting a Broken Link

Open an issue with the title `[broken] Name of entry` and the section it's in.

## Proposing a New Section

Open an issue first to discuss before adding. New sections need at least 3 quality entries to justify existence.
